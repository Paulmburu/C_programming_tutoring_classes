# Reserved Keywords

Here is a table of every reserved keyword in the C language,
for reference:
  ------------------------------
| auto    | if       | break    |
| int     | case     | long     |
| char    | register | continue |
| return  | default  | short    |
| do      | sizeof   | double   |
| static  | else     | struct   |
| entry   | switch   | extern   |
| typedef | float    | union    |
| for     | unsigned | goto     |
| while   | enum     | void     |
| const   | signed   | volatile |
  ------------------------------
